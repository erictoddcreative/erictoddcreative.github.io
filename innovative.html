Search Page
Redesign

This page design for Innovative Interfaces was created as a search page for a library, to be used both online and on kiosks in the library itself. The page needed to include different sections required by the client and is going to be customizable beyond the 'Recommended For You' page to the libraries themselves.

Kids Search
Page Redesign

This was created in addition to the search page above as a kids and young adults site that can be selected. This will restrict any content not suited for young audiences, and will primarily be used on kiosks in libraries themselves but will also be available online. The focus is on categories so younger users can easily find something they're interested in.

Innovative Backend Solution

This page layout was created as a backend module for libraries to edit their site and the links on the site. This was created following paradigms on the rest of the site developed by coworkers at Dreamten.
