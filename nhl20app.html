The UX
Explained

This is a low-res screenshot of the Adobe XD file created for the prototpe of the app interface. The UX for this application was focused on speed. The UI doesn't use many photographs, except for essential elements such as the card and pack images. The layout focuses on allowing the user to get to any page with ease, simply sliding out the menu from the side. All interactions are intuitive touch actions, allowing users to open information on cards with a tap, or view the current packs with a slide.

The UI
Explained

In the theme of the app I used the purple to pink gradient featured on the standard cover for the game. You can see some of the individual pages I created below. When creating the pages for the product I focused on which pages would showcase the best features of the app. I included a tab for the core activities an Ultimate Team player would engage in apart from actually playing the game. A lot of the UI elements are based on the visuals in the actual game, such as the player stats page and pack icons.
